STM32F7 Crypto Hardware Acceleraion Examples

Copyright 2015 STMicroelectronics


STM32 Crypto hardware acceleration Library provides a rich set of examples covering the main features of each cryptographic algorithm. All examples are available in source file format and provided with preconfigured project for the supported board. Expected output data are added in almost examples in order to check that the examples work correctly. Expected data are taken from cryptographic standard. Please refer to readme.txt in each example for more details about cryptographic standard used.


How to proceed?

  • In order to make the example work, you have to do the following:
    • Open the example using your preferred toolchain
    • Rebuild all files and load the image into target memory
    • Run the example by following the readme.txt instructions
